# TY - Medical AI Research Projects

A collection of 22 machine learning applications in healthcare, each focusing on a different medical specialty and dataset.

## Projects

Each project is accessible at `joshdrobert.github.io/ty/[project-name]`:

1. **Rucha** - Skin cancer classification from dermoscopy images
2. **Sammy** - Endovascular procedure success prediction
3. **Shalini** - Stem cell differentiation prediction
4. **Maansi** - Multi-disease chest X-ray classifier
5. **Hannah** - Drug response prediction
6. **MaryGrace** - Fetal health classification
7. **Charmaine** - Diabetic retinopathy detection
8. **Fouzul** - PAD severity prediction
9. **Amirah** - Neuromuscular disorder classification
10. **Neha** - Neonatal sepsis prediction
11. **Annabelle** - Pediatric bone fracture classification
12. **Timi** - Stroke type prediction
13. **Eric** - Motor intent classification from EEG
14. **Cyril** - HCC recurrence risk prediction
15. **Chris** - Alzheimer's disease stage classification
16. **Stephan** - Motion sickness severity prediction
17. **Sabour** - Bladder cancer recurrence prediction
18. **Musa** - Nanoparticle drug delivery efficiency
19. **Aaron** - Acute kidney injury prediction
20. **Josh** - Cardiac valve abnormality classification
21. **Jenna** - Opioid overdose risk prediction
22. **Taha** - Coronary artery disease prediction

## Setup

This repository is designed to be hosted on GitHub Pages. Each project is self-contained and runs entirely in the browser using client-side JavaScript and TensorFlow.js.
