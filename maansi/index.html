About This Model

This deep learning model classifies chest X-ray images into three categories: Normal, Pneumonia, and COVID-19. The model uses a convolutional neural network trained on a large dataset of chest X-rays to assist in rapid diagnosis.

Dataset: Chest X-Ray Pneumonia Dataset
